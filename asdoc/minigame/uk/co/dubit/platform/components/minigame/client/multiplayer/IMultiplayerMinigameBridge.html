updateGameState()method 
public function updateGameState(attribute:String, value:Object):void

Updates the shared gamestate. Any attribute updated will notify other connected clients to the change. The value must be a JSON transmittable object - i.e. String, number, boolean, array. Complex objects will not transmit correctly. For example, this could be called when a 'piece' in a game board is moved
